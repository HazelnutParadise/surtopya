# 歡迎來到 Surtopya——你是怎樣的問卷人？

**問卷描述**：不管你是第一次接觸問卷工具，還是已經是老手，我們都想先認識你一下。填完大約 3 分鐘，你也會更清楚 Surtopya 能為你做什麼。

---

## 第 1 頁
**標題**：歡迎來到 Surtopya，先聊兩句
**描述**：每個人用問卷的方式都不太一樣，花一點時間告訴我們你的情況，我們會更知道怎麼幫你。

**Q1** 單選｜你主要的身份是？
- 學生（含研究所）
- 學者 / 職業研究員（教授、博士後、研究機構等）
- 行銷 / 商業
- 設計師 / 產品 / UX
- 教育工作者（中小學教師、培訓機構等）
- 其他

**Q2** 多選｜你過去用問卷主要是做什麼？
- 學術研究
- 產品 / 用戶研究
- 市場調查 / 消費者研究
- 員工 / 社群意見蒐集
- 民調 / 公眾意見調查
- 還沒用過
- 其他

---

## 第 2 頁
**標題**：你之前做問卷的經驗是？
**描述**：不管是第一次，還是已經用過其他工具，都沒關係——說說你的狀況。

**Q3** 多選｜你用過哪些問卷工具？
- Google 表單
- SurveyMonkey
- Typeform
- 其他
- 沒用過 → **跳到 Q5**

**Q4** 多選（限選 2）｜用這些工具時，你最大的困擾是？
- 很難找到受眾
- 只能一次性發送，之後沒人填了
- 數據不好整理
- 費用太高
- 沒有特別的困擾

**Q5** 多選｜問卷填完後，你希望回應可以做什麼？
- 導出來自己分析
- 分享結果給他人看
- 長期持續蒐集，不只是一次快照
- 放到市場上讓其他人也能用

---

## 第 3 頁
**標題**：你今天想從哪裡開始？
**描述**：沒有標準答案，就算只是來逛逛也完全 OK。

**Q6** 單選｜你最想先嘗試的功能是？
- 建立一份問卷
- 探索別人發的問卷
- 看看數據集市場
- 只是來看看

**Q7** 單選｜你有沒有現成的問卷想搬過來？
- 有，已經在別的地方做好了
- 有想法但還沒開始
- 沒有特定計畫

---

## 結尾頁
**標題**：謝謝你告訴我們這些
**內文**：我們很高興你在這裡。根據你的回答，建立第一份問卷可能是個不錯的起點——它會自動成為一個可以被持續找到的頁面，不需要你反覆推送。

**按鈕**：建立我的第一份問卷 ／ 先逛逛其他問卷
